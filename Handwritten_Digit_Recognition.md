Handwritten Digit Recognition
File: digit_recognition.ipynb

Objective
To build a machine learning model capable of classifying images of handwritten digits (0–9).

Dataset Overview
This project utilizes the built-in "load_digits" dataset from Scikit-learn.

	1. Total Samples: 1,797 digit images.

	2. Image Properties: Each image is 8x8 pixels (64 features per sample).

	3. Classes: 10 distinct classes representing digits 0 through 9.

Implementation Details

	1. Model: Logistic Regression.

	2. Data Split: 80% Training and 20% Testing.

Key Results
	* The model achieved an overall accuracy of 97.5% in classifying the handwritten digits.
